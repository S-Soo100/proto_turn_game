## 1. 시장 개요

- 2025년 전 세계 모바일 게임 다운로드 **59.2억 건** (전년 대비 +4%)
- 퍼즐 장르는 2019년 이후 매년 **수익 3위** 유지
- 퍼즐 & 워드 게임은 심플한 게임플레이 루프, 무한 리플레이성, 창의적 메카닉으로 2026년에도 시장 지배 중

## 2. 2025-2026 인기 퍼즐 게임 분석

### 2.1 Block Blast (블록 퍼즐) — 다운로드 1위

| 항목 | 내용 |
|---|---|
| 개발사 | Hungry Studio |
| 다운로드 | **3.56억** (2025년 전 세계) |
| DAU | 1,400만 |
| 장르 | 블록 퍼즐 (테트리스 변형) |

- 격자에 블록 조각을 배치하여 가로/세로 줄을 완성하면 제거
- 테트리스의 향수 + 현대적 캐주얼 UX의 결합
- 시간 제한 없이 전략적으로 배치하는 방식이 핵심

### 2.2 타일 매치 / 마작 솔리테어 — 최대 성장 장르

| 항목 | 내용 |
|---|---|
| 성장률 | 마작 솔리테어 카테고리 **+189%** YoY |
| 다운로드 | 마작 솔리테어 **2.58억**, 타일 매치 전체 **5.03억** |
| 대표작 | Vita Mahjong (1.22억 DL, 7위), Tile Explore, Tile Club |

- 쌍을 이루는 타일을 찾아 제거하는 매칭 메카닉
- 짧은 세션, 낮은 진입 장벽, 높은 중독성

### 공통 성공 요인
1. **심플한 규칙** — 5초 안에 이해 가능
2. **무한 리플레이** — 레벨 생성 or 점수 경쟁
3. **짧은 세션** — 1판 1~3분
4. **시각적 만족감** — 줄 제거, 타일 매칭 시 피드백 애니메이션

## 5. 추천 후보 TOP 3

### 1순위: 블록 퍼즐 (Block Blast 류)
- **이유**: 2025년 다운로드 1위, 구현 난이도 적당, 기존 격자 기반 렌더링 경험 활용 가능
- **핵심 기능**: 블록 배치 → 줄 완성 → 제거 → 점수, 하이스코어 기록
- **예상 작업량**: 게임 로직 + 보드 컴포넌트 + 점수 시스템

### 2순위: 타일 매치 (마작 솔리테어 류)
- **이유**: 189% 성장의 급상승 장르, 시각적 완성도 높음
- **핵심 기능**: 타일 레이어 쌓기, 쌍 매칭, 레벨 생성
- **예상 작업량**: 타일 렌더링(레이어 표현) + 매칭 로직 + 레벨 생성기

### 3순위: 2048
- **이유**: 구현 가장 빠름, 검증된 중독성, 스와이프 UX
- **핵심 기능**: 4x4 격자 타일 합치기, 하이스코어
- **예상 작업량**: 비교적 적음 (슬라이드 로직 + 합치기 애니메이션)
