## 연출

### 시각 효과
1. 발동 시 화면 상단에서 댓글이 오른쪽→왼쪽으로 흘러가기 시작
2. 2~4초 간격으로 새 댓글 등장
3. 댓글은 반투명 배경 + 흰색 텍스트 (탄막 스타일)
4. 가끔 댓글이 돌 위치를 **살짝 가림** (간접 방해)
5. 상황에 따라 댓글 내용이 변함 (실패 시 조롱, 성공 시 시큰둥)

## 구현 가이드

### 데이터 구조
```typescript
{
  id: 'CR-GG007',
  name: '관객 야유',
  trigger: 'during-play',
  minRound: 3,
  probability: 0.40,
  execute: (state: GonggiState) => ({
    type: 'ui-overlay',
    animation: 'danmaku-comments',
    message: null, // 댓글이 곧 메시지
    duration: 'until-round-end',
  }),
}
```

### 필요한 애니메이션
- 댓글 이동: CSS `@keyframes slide` — `translateX(100vw)` → `translateX(-100%)` (4초)
- 댓글 스타일: 반투명 배경(`rgba(0,0,0,0.5)`) + 흰색 텍스트 + 둥근 모서리
- y 위치: 랜덤 (화면 상단 20~80% 범위, 돌 영역 일부 침범)
- 등장: Framer Motion `opacity: [0, 1]` (0.2초)

### 댓글 컴포넌트 구조
```typescript
interface DanmakuComment {
  id: string
  text: string
  y: number      // 화면 높이 비율 (0.2~0.8)
  speed: number  // 이동 속도 (3~5초)
  color: string  // 텍스트 색상
}
```

### 참고 패턴
- 니코니코/빌리빌리 탄막 댓글 UI
- CSS animation `linear` + `forwards` 패턴

## 메모

- 게임에 직접적 영향이 없는 유일한 변칙 룰 — 하지만 심리적 데미지가 큼

- 한국어 인터넷 밈/말투를 활용하면 몰입감 상승
- 향후: 플레이어 닉네임을 댓글에 포함하는 변형 ("OO님 못하시네 ㅋ")
