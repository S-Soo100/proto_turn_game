## 요약

> 화면 뒤집기(screen-flip) 변칙 룰을 완전히 삭제한다.

## 상세 설명

화면 뒤집기 룰은 단계 전환 시 화면을 180도 회전시키는 변칙이다. 게임 경험상 불필요하다고 판단되어 삭제한다.

삭제 대상:
- 룰 정의 파일
- GonggiBoard.tsx 내 screen-flip 관련 CSS/로직 (`isFlipped` 상태, Container rotate 등)
- 테스트 코드 내 screen-flip 관련 assertion

## 구현 가이드

### 수정할 파일
| 파일 | 변경 내용 |
|---|---|
| `src/lib/game-logic/chaos-rules/screen-flip.ts` | **삭제** |
| `src/components/game/GonggiBoard.tsx` | screen-flip import 제거, `isFlipped` 상태 제거, Container CSS rotate 로직 제거 |

| `src/components/game/GonggiBoard.test.tsx` | screen-flip 관련 테스트 제거 |
