## 요약

> 공기놀이의 게임 흐름에 "돌 선택(select)" + "손에 올리기(hold)" 단계를 추가하여, 실제 공기놀이처럼 플레이어가 돌을 직접 골라 손에 올리고 던지는 경험을 제공한다. hold 시 z축 시각 표현으로 돌이 바닥에서 떠오르는 효과를 구현한다.

## 상세 설명

### 현재 흐름 (문제)

```
scatter → toss → pick → catch → (반복 또는 stage-clear)
```

- `toss`: 시스템이 자동으로 돌 하나를 선택하여 던짐
- 플레이어는 어떤 돌이 던져지는지 모르고, 돌을 선택하는 인터랙션이 없음
- 돌을 손에 쥐는 "대기" 상태가 없어서 던지기 전 확인 불가

### 개선 흐름

```
scatter → select → hold → toss → pick+catch → (반복 또는 stage-clear)
              ↑ 신규     ↑ 신규
```

### 단계별 상세 + z축 연동

#### 1. scatter
- 게임 로직이 돌 5개의 x,y 좌표 생성 (15~85% 범위)
- 모든 돌 z=0 (바닥)
- CSS transition으로 이전 위치 → 새 위치 이동 (0.3초)
- **화면**: 돌 5개가 보드 전체에 흩어짐

#### 2. select (신규 phase)
- **메시지**: "던질 돌을 골라주세요"
- **인터랙션**: 바닥 돌 탭
- **선택 피드백**:
  - 선택된 돌: 금색 테두리 glow + **z=0.15로 미세하게 떠오름** (scale 0.95, y-offset -12px, 그림자 약간 확산)
  - "이 돌을 올릴까요?" 느낌
- **탭 변경**: 다른 돌 탭하면 기존 돌 z=0으로 복귀, 새 돌 선택
- **자동 진행**: 선택 완료 후 0.3초 뒤 hold 단계로 전환
- **5단계(꺾기)**: select 스킵 → 바로 toss (전부 던지기)

#### 3. hold (신규 phase)
- **선택한 돌의 z축 이동**: z=0.15 → z=0.3 (transition 0.3초)
  - scale: 0.95 → 0.90
  - y-offset: -12px → -24px
  - 그림자: 중간 크기, 약간 흐림
- **시각적 위치**: 돌이 바닥에서 들어올려져 "손에 쥔" 느낌
- **손 영역 UI**: 보드 상단에 반투명 "🤚 손" 아이콘 + 선택된 돌 표시
- **버튼**: "던지기 🫴" 활성화
- **돌 변경**: hold 중에도 다른 돌 탭 가능 → 기존 돌 z=0 복귀, 새 돌 z=0.3
- **화면 구성**:

```
┌─────────────────────┐
│  🤚 + 🟡 (z=0.3)   │  ← 손 영역 (들어올린 돌)
│                     │
│  🔴    🔵           │  ← 바닥 돌들 (z=0)
│     🟢     🟣       │
│                     │
│    [ 던지기 🫴 ]     │  ← 액션 버튼
└─────────────────────┘
```

#### 4. toss (기존 수정)
- hold 상태에서 "던지기" 버튼 → 돌이 z=0.3 → z=1.0 → z=0 포물선
- 포물선 상세는 T-GG011에서 구현
- 나머지 돌은 바닥(z=0)에 고정

#### 5. pick (기존 수정)
- 던진 돌이 z=1.0에서 하강 시작 → 바닥 도달 전까지가 pick 시간
- 바닥 돌을 스와이프하여 필요 개수만큼 선택
- 시간 제한은 T-GG011에서 구현

#### 6. catch (기존 수정)
- 던진 돌이 z≈0 근처에서 "잡기" 버튼
- 타이밍 판정은 T-GG011에서 구현

### GamePhase 타입 확장

```typescript
export type GamePhase =
  | 'scatter'
  | 'select'      // 신규: 던질 돌 선택
  | 'hold'        // 신규: 손에 올리기 (던지기 대기)
  | 'toss'
  | 'pick'
  | 'catch'
  | 'stage-clear'
  | 'success'
  | 'failed'
```

### 순수 함수 추가 (gonggi.ts)

```typescript
// 돌 선택 — floor 돌 하나를 select 상태로
selectStone(state: GonggiState, stoneId: number): GonggiState

// 손에 올리기 — select된 돌을 hold 상태로 (z=0.3)
holdStone(state: GonggiState): GonggiState

// startToss 수정 — hold 상태 돌을 toss로 전환 (기존: 자동 선택 → 변경: hold 돌 사용)
startToss(state: GonggiState): GonggiState
```
