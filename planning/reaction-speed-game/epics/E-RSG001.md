## 목표

120초 동안 축소 원(타겟 서클)을 클릭하여 점수를 모으는 반응속도 게임 MVP를 구현한다. 게임 로직, 보드 UI, 점수/콤보 시스템, Supabase leaderboard 연동까지 포함.

## 배경

현재 프로젝트는 턴제 보드 게임(틱택토, 오목)만 제공. 반응속도 기반의 실시간 솔로 게임을 추가하여 게임 허브의 장르 다양성을 확보한다. 기존 격자 기반이 아닌 자유 배치 + 타이머 기반의 새로운 게임 패턴.

## 범위

### 포함
- 게임 로직: 타겟 스케줄 생성, 점수 계산, 콤보 시스템, 등급 판정
- 보드 UI: osu! 스타일 축소 원, HUD(타이머), 상태바(점수+콤보)
- 타겟 서클 컴포넌트: Normal/Speed 2종, 축소 애니메이션, 클릭 피드백
- 결과 화면: 상세 통계 (총점, 클릭수, 미스수, 적중률, 최대콤보, 등급 비율)
- DB: leaderboard 테이블 신규 생성 + game_types에 행 추가
- 랭킹: 게임 선택 화면에 Top 10 표시 (순위 + 닉네임 + 점수 + 적중률)
- 홈 화면: 반응속도 게임 카드 추가

### 제외
- PvP 실시간 대전 모드
- 타겟 서클 종류 추가 (Bonus, Trap 등)
- 일일 도전 / 시즌 시스템

## 기술 고려사항

- DB 변경 필요: `leaderboard` 테이블 신규 생성 + `game_types`에 행 추가 (SQL Editor 실행)
- 기존 게임과 다른 패턴: 턴제가 아닌 실시간 타이머 기반 → `requestAnimationFrame` 또는 `setInterval` 필요
- 타겟 서클 동시 다수 렌더링 → 성능 고려 (최대 동시 5~6개)
- 축소 원 애니메이션: CSS animation 또는 Framer Motion
- 모바일 터치 이벤트 처리: 터치 영역과 타겟 서클 히트 판정
