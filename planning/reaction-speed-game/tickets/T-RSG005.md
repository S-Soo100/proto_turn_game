## 요약

반응속도 게임을 기존 앱에 통합한다. 홈 화면에 게임 카드 추가, 게임 선택 화면(랭킹 + 시작 버튼), 결과 화면(상세 통계 + 저장), 게임 스토어 연동.

## 상세 설명

### 홈 화면 (HomePage)
- `ACTIVE_GAMES` 배열에 반응속도 게임 카드 추가
- 솔로 전용 → 바텀시트에서 "친구와 대전" 대신 바로 게임 선택 화면으로 이동
- 카드 디자인: 반응속도 게임 아이콘/이모지 + 간단한 설명

### 게임 선택 화면
- Top 10 랭킹 리스트 (순위 + 닉네임 + 점수 + 적중률)
- 내 최고 기록 표시 (있으면)
- "시작" 버튼 → 게임 화면으로 전환

### 결과 화면 (게임 종료 후)
- 바텀시트 또는 오버레이로 표시
- 통계: 총점, 클릭 수, 미스 수, 적중률 (%), 최대 콤보, Perfect/Great/Good/OK 개수/비율
- 랭킹 순위 표시 (Top 10 진입 시 축하 메시지)
- "확인" → leaderboard에 저장 + 게임 선택 화면으로 복귀
- "다시 하기" → 바로 새 게임 시작

### gameStore 연동
- `GameTypeId`에 `'reaction-speed-game'` 추가
- 또는 별도 `reactionSpeedStore` 생성 (기존 턴제 게임과 패턴이 많이 다르므로)
