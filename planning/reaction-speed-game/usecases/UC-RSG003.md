## 사전 조건

- 플레이어가 로그인 상태
- 홈 화면에서 반응속도 게임을 선택한 상태

## 기본 시나리오 (Happy Path)

1. 플레이어가 게임 선택 화면에 진입한다
2. 시스템이 leaderboard에서 Top 10을 조회한다
3. 랭킹 리스트가 표시된다 (순위 + 닉네임 + 점수 + 적중률)
4. 플레이어 본인의 최고 기록이 있으면 별도로 강조 표시된다
5. 플레이어가 랭킹을 확인한 뒤 "시작" 버튼으로 게임을 시작한다

## 대안 시나리오

### A1: 랭킹 데이터 없음
- 조건: 아직 아무도 플레이하지 않아 leaderboard가 비어있음
- 흐름:
  1. "아직 기록이 없습니다. 첫 번째 도전자가 되어보세요!" 메시지 표시
  2. "시작" 버튼은 정상 동작

### A2: 본인 기록이 Top 10에 포함
- 조건: 플레이어의 최고 기록이 Top 10 안에 있음
- 흐름:
  1. 해당 행이 하이라이트 처리 (배경색 또는 아이콘)
  2. "내 기록" 라벨 표시

## 예외 시나리오

### E1: 랭킹 조회 실패
- 조건: 네트워크 오류로 leaderboard 조회 실패
- 시스템 반응: "랭킹을 불러오지 못했습니다" 메시지 + 게임 시작은 가능

## 검증 방법

- [ ] 게임 선택 화면 진입 시 Top 10 랭킹이 표시되는지 확인
- [ ] 랭킹이 점수 내림차순으로 정렬되어 있는지 확인
- [ ] 닉네임, 점수, 적중률이 올바르게 표시되는지 확인
- [ ] 본인 기록 하이라이트 확인
- [ ] 빈 leaderboard 시 안내 메시지 확인
